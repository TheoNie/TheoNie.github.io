hexo博客快速搭建(入门篇)

  • 我的博客搭建日记

一直都想搭建一个博客，但是总是半途而废，兜兜转转终于决定要坚持记录下来自己学习的一些轨迹，结构化起来，害怕以后会忘。
最近发现hexo很好用，就记录一下自己的搭建过程，当然也比较简单，所以只是入门篇，目标是搭载到github上。进阶的一些东西也会慢慢的研究下去。

1. 注册一个github的账号，然后新建一个仓库，仓库的名字一定要是username.github.io，比如我的就叫TheoNie.github.io.

2. 下载node.js，官网下载一个最新版就好了，我用的是Windows，一路可以默认安装。

3. 下载git，也是可以默认安装。

4. 安装hexo。使用git CMD，执行npm install -g hexo.

5. 初始化。在你想使用的本地博客文件夹下使用 git bash ，执行hexo init.

6. 配置ssh。这个可以参考官网的GitHub Help - Generating SSH Keys

7. 然后可以先安装一个hexo的主题，我使用的主题是这个hexo-theme-yilia，使用说明在主页就有，安装其他的主题的方法也是类似的。

8. 部署到github。编辑hexo的_config.yml文件，这个文件里是关于hexo的基本配置，如果我们要部署到github上的话，要把delay这一个属性修改成这样，注意要用自己仓库的名字。

注意这里的type有可能出现错误提示Deployer not found: github，然后可以把type改为git，然后在git cmd下执行npm install hexo-deployer-git --save.

9. 执行hexo g

10. 执行hexo d

11. 现在可以在http://username.github.io/中看到你的博客了。

12. 以后还想再上传新博客的时候，可以执行hexo new newtitle,再新建一个新文章，文章使用markdown语法。

13. 再次执行第九步和第十步，就可以看见新文章了。

hexo支持本地预览，可以使用hexo server指令在localhost中预览再发布
